# Ford Admin

## IDE
 - Visual Studio Code

## System requirements

 - NodeJs v20.11.1 or higher

 - Yarn v1.22.21 or higher

## Web Config
 - `cp .env.example .env`

 - update .env file

 - update environment.d.ts if you want to use typescript

## Install dependencies
 - `yarn`

## Start web
 - `yarn dev`
 - run at http://localhost:3000
